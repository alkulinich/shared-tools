# New Project: Initialize CLAUDE.md

Create CLAUDE.md for the new project with context from PLAN.md and project setup.

## Instructions

0. **Track command:** `shared/scripts/set-current-command.sh new-project`

1. **Read context:**
   - Read `PLAN.md` from the project root
   - Read any existing `CLAUDE.md` (to avoid overwriting)
   - Identify the tech stack, project structure, and key conventions from the plan

2. **Create `CLAUDE.md`** with sections:
   - Project overview (1-2 sentences)
   - Tech stack
   - Project structure (key directories)
   - Development setup (how to run, test, build)
   - Key conventions and patterns
   - Reference to `PLAN.md` for full implementation plan

3. **Keep it concise** — CLAUDE.md should be a quick reference, not a copy of the plan.

4. **Confirm:**
   ```
   CLAUDE.md created. Run /new-project:4-plan-issues to break the plan into GitHub issues.
   ```
